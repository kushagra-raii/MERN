Learn About Quotation and Citation Tag (blockquote q cite address abbr).

The HTML <blockquote> element defines a section that is quoted from another source.(Usually indents elements)

The HTML <q> tag defines a short quotation.( insert quotation marks around the quotation)

The HTML <cite> tag defines the title of a creative work(italicize)

The HTML <address> tag defines the contact information for the author/owner of a document or an article.(italicize)

The HTML <abbr> tag defines an abbreviation or an acronym(useful info Search Engine,Browser)
